# NodeJS Playground

## Getting Started

```bash
# install dependencies
yarn

# prepare the repo
yarn copyfiles

# build the app the first time
yarn build

# start the app
yarn start
```

## How to run the Docker Image

```bash
docker-compose up --build
```

## Build and Deploy Docker Image

```bash
docker-compose build
```
